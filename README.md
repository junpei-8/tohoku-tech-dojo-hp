# 東北 TECH 道場 Web サイト

## 必要環境

- Node
- Docker
- Make
- Google cloud SDK
- Firebase tools

## 開発環境立ち上げ

```
yarn dev
```

## デプロイ

```
make deploy
```

＊ 事前に `setup-gcloud` で Google Cloud SDK の設定を実行している必要があります。
